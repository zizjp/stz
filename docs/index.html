指標の説明

  • 年率リターン（CAGR）：複利年率換算での平均収益率。S&P500は2021年末基準で過去5年累計約50％増（年約8.5％）
  • 年率ボラティリティ：価格変動の激しさ。数値が大きいほどリスクが高い。暗号資産は株式の約3～5倍
  • 最大ドローダウン（DD）：ピークからの最大下落率。投資家心理と資金管理に与える影響が大きい
  • 95％VaR：95%の確率で発生しない推定最大損失率（期間同一前提）。リスク許容度と資金配分の基準に用いる

※略記：CAGR＝複利年率、DD＝最大ドローダウン、VaR＝バリュー・アット・リスク

NISA活用と税制最適化

新NISA：生涯1,800万円 年360万円（つみたて120／成長240） 非課税・無期限

NISA枠の優先配分

  1. 高成長資産（米国株ETF、FANG+系）
  2. 分配金発生資産（高配当ETF）
  3. 金ETF・国際資産（為替差益含む）
  4. 長期保有予定の成長株

課税口座での留意点

  • 暗号資産は損益通算不可（原則雑所得扱い）
  • 為替差益も約20.315%課税対象
  • 売却益に約20.315%課税（所得税＋住民税）
  • 損益は口座間で通算不可

リバランス戦略とルール

方針：年1回の定期＋目標比から±10%乖離で機動的に調整（新規積立の配分変更を最優先）

売買の優先順位（原則）

  1. 新規積立の配分変更で乖離を徐々に是正
  2. 不足資産の追加買付（下落局面は優先）
  3. 超過資産の部分売却（NISA保有は極力維持／課税口座の含み益・含み損を活用）
  4. 必要に応じ為替ヘッジ・現金比率でクッション調整

口座別の留意点

  • NISAと課税口座の売却順序：課税口座で調整が原則（損出し・益出しで税最適化）。NISA保有は長期温存し、どうしても必要な場合のみ一部売却。
  • 売買コストに基づく閾値設定：手数料＋スプレッド＋想定税負担の合計が0.5〜1.0%を上回るまでは 積立配分の変更だけで吸収（デッドバンド運用）。
  • 暗号資産の課税：売却益は原則 損益通算不可。税率は制度に依存（便宜上約20%前提で試算、最新制度は要確認）。

※税制は2025年11月時点の整理。最新の取り扱いは所轄・税理士へ確認。

実務フロー（チェックリスト）

  • ① 現在配分→目標配分の乖離率を算出（全体／資産別）
  • ② 積立の配分変更で是正（目安2〜6か月で収斂）
  • ③ それでも乖離が大きい場合のみ売買（課税口座優先／損出し・益出しで税最適化）
  • ④ 売却余剰は金・現金へ一時避難→段階的再投資
※±10%は相対変化基準（例：35%→31.5%で−10%）。売買は手数料・税を考慮して分割実行。

KPIダッシュボードと次のアクション

いつも見る数字（KPI）：週次更新／月次レビュー
  • 目標までの進み具合：現在資産が1億円ラインに対して何％進捗か
  • 値動きの荒さ（年率ボラ）：全体・資産別の“揺れ幅”。大きいほど上下に振れやすい
  • 今年の最安更新幅（YTD最大DD）と最悪ライン（95%VaR）：今年どこまで落ちたか／通常ここまでは悪化しない目安
  • 配分のズレ（配分乖離率）：実際の比率が目標配分から外れている度合い
  • 暗号/金の30日・90日ブレ：直近1か月/3か月の標準偏差で荒れ具合を把握
  • 株との連動（相関の変化）：株下落に巻き込まれやすくなっていないか

アラート条件（動く基準）

  • 配分が目標から±10％以上：積立配分の変更でリバランス検討
  • YTD最大DDが▲15％超：リスク低減（現金/金を増やす・暗号を減らす・レバ回避）
  • 暗号の月間ボラ急騰：30日ボラが直近平均比+50％以上または相関急上昇 → 配分見直し
  • 年初来リターンが目標から▲30％超：戦略見直し（積立増額／リスク配分調整／期間再設定）
